Dependencies

This repository contains a Angular v5+ rich mentions component. It allows for customizable mention display and custom searching. The only required dependencies are:

  • Angular (requires Angular version 6 or higher, tested with 5.0.0)
Supported browsers

We strive to support the same browsers and versions as supported by Angular. Check browser support notes for Angular.

Our code is automatically tested on all the supported browsers.
